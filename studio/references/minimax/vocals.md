## Backing Vocals

### Presence Levels

| Level | Description | When to Use |
|-------|-------------|-------------|
| **High** | Dense harmonies, choir-like, constant | Gospel, anthems, epic tracks |
| **Medium** | Tasteful harmonies on chorus, light doubling | Pop, R&B, rock |
| **Low** | Sparse "ooh"s or "aah"s, fills only | Intimate tracks, acoustic, lo-fi |
| **None** | Solo voice only | Spoken word, minimal, raw |

### Backing Vocal Types

| Type | Description |
|------|-------------|
| **Choir** | Full ensemble, layered voices |
| **Harmonies** | 2-3 voices in harmonic intervals |
| **Doubling** | Same melody for thickness |
| **Ad-libs** | Improvised responses, fills |
| **Call and response** | Lead and backing alternate |
| **Oohs and aahs** | Wordless vocal pads |
| **Gang vocals** | Group unison, anthemic |
| **Whisper layers** | Ghostly doubled whispers |

|------|-------------|----------|
| **Cathedral / Large hall** | Massive, expansive | Epic ballads, orchestral, ambient |
| **Medium room** | Natural, balanced | Pop, rock, general use |

## Vocal Direction Tags (for Lyrics Field)

Use these inline in lyrics to guide vocal delivery at specific moments:

```
[Soft vocals]        — Gentle, restrained
[Whispered]          — Hushed, intimate
[Spoken word]        — Rhythmic speech
[Powerful vocals]    — Full voice, commanding
[Belting]            — Sustained high energy
[Falsetto]           — High, airy
[Harmonies]          — Layered voices
[Ad-libs]            — Improvised fills
[Humming]            — Closed-mouth melody
[Vocalizing]         — "Ooh", "aah", wordless
[Call and response]  — Lead and backing alternate
[Growling]           — Aggressive texture
[Raspy]              — Rough, lived-in
```
