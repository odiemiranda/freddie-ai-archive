# MiniMax Music — Production & Arrangement Reference

Professional production terminology and techniques for the MiniMax Music Style field. MiniMax-2.5 achieves professional-grade mixing quality, so detailed production language yields excellent results.

---

## Arrangement Elements

### Core Instruments by Genre

| Genre | Foundation | Character | Texture |
|-------|-----------|-----------|---------|
| **Pop** | Programmed drums, bass synth | Synth pads, guitar | Vocal layers, percussion |
| **Rock** | Live drums, electric bass | Electric guitar, keys | Backing vocals, FX |
| **Hip-Hop** | 808 kick, programmed drums | Sampled chords, bass | Vocal chops, FX |
| **R&B** | Soft drums, warm bass | Rhodes, guitar | Pads, vocal harmonies |
| **Jazz** | Brush drums, upright bass | Piano, sax, trumpet | Vibraphone, guitar |
| **Folk** | Minimal percussion | Acoustic guitar, fiddle | Harmonica, mandolin |
| **Electronic** | Drum machine, sub-bass | Synth leads, arps | Pads, risers, FX |
| **Country** | Acoustic drums, bass | Acoustic/electric guitar | Pedal steel, fiddle |
| **Classical** | Timpani, percussion | Strings, brass | Woodwinds, harp |
| **Soul/Funk** | Tight drums, slap bass | Horns, organ | Clavinet, wah guitar |

### Instrument Limit

Keep to **3-5 instruments** per prompt for clarity. Specify playing technique for precision:

| Instrument | Techniques |
|------------|-----------|
| **Guitar** | Fingerpicking, strumming, palm-muted, arpeggiated, power chords, slide |
| **Piano** | Chord stabs, arpeggiated, rolling chords, sparse single notes, percussive |
| **Bass** | Walking, slap, fingerstyle, picked, sub-bass, fretless |
| **Drums** | Boom-bap, four-on-the-floor, shuffle, brush, programmed, breakbeat |
| **Strings** | Legato, pizzicato, tremolo, staccato, sustain |
| **Synth** | Pad, lead, arpeggio, pluck, bass, sweep |
| **Brass** | Staccato stabs, legato lines, muted, solo |

|---------|-------------|-----------------|

| Setting | Description | Prompt Fragment |

| **Layered depth** | Front, mid, and back elements | "layered depth, vocals forward, guitars mid, pads distant" |
| **Deep and spacious** | Lots of distance between elements | "deep spacious mix, elements placed at varying distances" |

## Dynamics Processing

### Compression Styles

| Style | Effect | Use For |
|-------|--------|---------|
| **Heavy compression** | Loud, punchy, consistent | Rock, pop, hip-hop, EDM |
| **Moderate compression** | Controlled but natural | General use, R&B, folk |
| **Light compression** | Mostly natural dynamics | Jazz, classical, acoustic |
| **Side-chain compression** | Pumping effect, bass ducks | EDM, dance, modern pop |
| **Parallel compression** | Blended natural + crushed | Rock drums, vocals |
| **Multiband compression** | Frequency-specific control | Mastering, polished mixes |

### Dynamic Range

| Setting | Description | Prompt Fragment |
|---------|-------------|-----------------|
| **Very compressed** | Loud, consistent, radio-ready | "heavily compressed, radio-ready loudness" |
| **Moderately dynamic** | Some variation, polished | "moderate dynamics, polished but natural" |
| **Highly dynamic** | Whisper to shout, cinematic | "wide dynamic range, from whisper to full power" |
| **Dynamic contrast** | Quiet sections vs loud sections | "dramatic dynamic contrast between sections" |

## Production Quick Recipes

### Polished Pop
```
Polished pop production, crisp drums, warm bass synth, bright synth pads, wide stereo mix with LCR panning, moderate compression for radio-ready loudness, vocals centered and forward
```

### Raw Indie
```
Raw indie rock production, live room drums, slightly overdriven bass, jangly guitars panned left and right, natural room reverb, minimal compression, vocals slightly set back
```

### Atmospheric Electronic
```
Atmospheric electronic production, programmed beats with sidechain compression, deep sub-bass, layered synth pads filling the stereo field, spacious reverb, distant vocals with heavy processing
```

### Warm Acoustic
```
Warm acoustic production, fingerpicked nylon guitar, minimal processing, natural room ambience, close-mic vocals with light plate reverb, organic and intimate
```

### Hard-Hitting Hip-Hop
```
Hard-hitting hip-hop production, booming 808 kick, crisp snare, rapid hi-hats, atmospheric pads, dark and punchy, heavy compression, vocals dry and forward
```
