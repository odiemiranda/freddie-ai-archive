# MiniMax Music — Moods & Emotions Reference

MiniMax Music uses **Emotion Tags** and **Scenario Tags** to guide generation. This file provides comprehensive mood categories for the Style field.

---

## Emotion Tags

### Happy + Uplifting

| Tag | Nuance | Pair With |
|-----|--------|-----------|
| Joyful | Pure happiness, celebratory | Major keys, bright instruments |
| Uplifting | Rising spirits, inspirational | Building dynamics, soaring vocals |
| Euphoric | Peak happiness, ecstatic | Big drops, anthemic choruses |
| Cheerful | Light, carefree | Acoustic guitar, hand claps |
| Playful | Fun, mischievous | Staccato, bouncy rhythms |
| Sunny | Warm, golden | Bright synths, light percussion |
| Triumphant | Victory, achievement | Brass, powerful drums |
| Grateful | Warm appreciation | Soft strings, gentle vocals |
| Whimsical | Lighthearted, quirky | Unusual instruments, playful melodies |
| Blissful | Deep contentment | Lush pads, flowing melodies |

**Example Style Prompt:**
```
Uplifting indie pop, bright and joyful, warm female vocals with wide smile, acoustic guitar strumming, hand claps, sunny morning feel, G Major, 118 BPM
```

## Scenario Tags

Scenario tags describe the listening context — where and when the listener experiences the music.

### Time of Day

| Scenario | Description | Mood Match |
|----------|-------------|------------|
| **Sunrise / Dawn** | First light, fresh start | Hopeful, gentle, awakening |
| **Morning** | Bright energy, optimism | Cheerful, energetic, warm |
| **Golden hour** | Warm afternoon light | Nostalgic, warm, romantic |
| **Sunset / Dusk** | Day ending, reflection | Bittersweet, reflective, warm |
| **Night / Midnight** | Late hours, solitude | Mysterious, intimate, dark |
| **3 AM** | Deep night, vulnerability | Raw, lonely, contemplative |

### Setting / Location

| Scenario | Description | Mood Match |
|----------|-------------|------------|
| **Morning café** | Coffee, gentle energy | Warm, relaxed, acoustic |
| **Rainy window** | Inside, rain outside | Cozy, melancholic, reflective |
| **Late-night drive** | Open road, city lights | Nostalgic, free, contemplative |
| **Rooftop at sunset** | Urban, elevated, open sky | Romantic, dreamy, expansive |
| **Beach bonfire** | Waves, warmth, friends | Warm, communal, relaxed |
| **Empty city street** | Urban isolation | Lonely, cinematic, noir |
| **Mountain trail** | Nature, fresh air | Peaceful, adventurous, free |
| **Underground club** | Dark, bass-heavy | Energetic, dark, hypnotic |
| **Concert arena** | Massive crowd, energy | Anthemic, powerful, electric |
| **Bedroom** | Private, personal space | Intimate, vulnerable, raw |
| **Library** | Quiet focus, knowledge | Contemplative, academic |
| **Studio apartment** | Small space, big dreams | Indie, personal, creative |
| **Old record store** | Vintage, discovery | Nostalgic, retro, eclectic |
| **Japanese garden** | Zen, minimalism | Peaceful, meditative |
| **Parisian streets** | Romance, elegance | Romantic, sophisticated |

## Mood Combinations

Layered emotions create more compelling music. Use "yet" or "with" to combine:

| Combination | Description | Genre Match |
|-------------|-------------|-------------|
| "Melancholic yet hopeful" | Sad with light breaking through | Indie, folk |
| "Dark but beautiful" | Ominous with gorgeous melody | Art pop, shoegaze |
| "Energetic but controlled" | Power with precision | Electronic, K-pop |
| "Nostalgic with warmth" | Looking back fondly | Lo-fi, indie |
| "Intimate yet expansive" | Close vocals, big space | Dream pop, ambient |
| "Fierce but vulnerable" | Strong exterior, soft core | Alt-R&B, rock ballad |
| "Mysterious with longing" | Unknown yearning | Dark pop, trip-hop |
| "Peaceful with depth" | Calm surface, complex underneath | Ambient, classical |
| "Playful but melancholic" | Fun with bittersweet edge | Indie pop, city pop |
| "Epic yet intimate" | Grand scale, personal story | Cinematic, orchestral pop |
