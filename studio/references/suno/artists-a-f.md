# Artists Reference: A-F

700+ artist styles for Suno AI prompts. Find an artist, use their signature sound.

## How to Use Artist Entries

Each artist entry is a comma-separated list of their signature elements. To build a prompt, combine the entry with the master formula:

```
[ARTIST STYLE ELEMENTS] + [INSTRUMENTS] + [MOOD] + [TECHNICAL SPECS]
```

### Example: From Entry to Prompt

**Entry:** `Adele: Soulful pop, powerhouse female vocals, piano-led, emotional ballads, cinematic`

**Tagged prompt:**
```
[Intro] Soft piano, building emotion [Verse] Soulful pop ballad, powerhouse female vocals, piano-led, cinematic strings, D Minor, 72 BPM, emotional depth, Adele influence [Chorus] [Powerful vocals] Soaring melody, full orchestral arrangement [Outro] [Fade Out]
```

**Natural-language prompt:**
```
Soulful pop ballad, powerhouse female vocals, piano and strings, cinematic emotional depth, Adele influence, D Minor, 72 BPM, heartbreak anthem
```

### Tips
- **Don't copy entries verbatim** — expand them with specific instruments, BPM, key, and mood
- **Combine 2 artists** for fusion: "Nujabes meets Massive Attack" = jazz-infused trip-hop
- **Add era specifics**: "Radiohead's OK Computer era" is more useful than just "Radiohead"
- **Use entries as departure points** — the entry tells you the artist's DNA, you add the creative direction

## Quick Prompt Formula

To use any artist:

```
[Instrumental] [Artist] style, [their key elements], [mood], [BPM/key if relevant] [No Vocals]
```

Example:
```
[Instrumental] Daft Punk style, French house groove, vocoder textures, disco-funk bass, robotic groove, 120 BPM [No Vocals]
```

## Artist DNA Extraction

When referencing an artist, identify:
1. **Signature instruments** - What defines their sound?
2. **Production style** - Lo-fi? Polished? Experimental?
3. **Emotional core** - What feeling do they consistently evoke?
4. **Era/aesthetic** - What visual/cultural world do they inhabit?
