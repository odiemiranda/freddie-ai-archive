## EDM Structure Templates

Codified build-drop-breakdown sequences for different electronic subgenres. Each template maps directly to Suno structure tags.

### Standard EDM (4-minute track)

**BPM:** 126-132 (sweet spot: 128)
**Key Instruments:** Festival lead synth, heavy sidechain kick, white noise risers
**Mood:** Euphoric, high-energy, anthemic, festival-ready
**Avoid:** Acoustic instruments, jazz chords, lo-fi texture, vinyl crackle, slow tempos

**Structure Template:**
```
[Intro] kick-less atmosphere + arp motif, filtered percussion
[Build] risers + snare roll, add bass movement
[Drop] full kick + bass, hook synth, strong groove
[Breakdown] remove kick, keep pad + vocal chop motif
[Build] tension returns, bigger risers, wider harmony
[Drop] second drop variation (new top line or bass rhythm)
[Outro] simplify elements, fade with motif
```

**Example Style Block:**
```
[Instrumental] EDM, festival house, 128 BPM, euphoric, supersaw lead, sidechain kick, white noise risers, snare build rolls, big room drop, clean mix, anthemic breakdown, wide stereo pads [No Vocals]
```

**Exclude Style:** `acoustic guitar, jazz, lo-fi, vinyl crackle, slow tempo`

**Advanced Settings:** Weirdness 25-40%, Style Influence 70-85%

### Deep House (loop-based)

**BPM:** 118-124 (sweet spot: 122)
**Key Instruments:** Warm analog pad, deep sub bass, shuffled hi-hats
**Mood:** Hypnotic, groove-driven, warm, late-night, sensual
**Avoid:** Hard drops, aggressive risers, distorted bass, festival energy, harsh synths

**Structure Template:**
```
[Intro] filtered kick + hi-hat shuffle, subtle pad wash
[Verse] full groove enters, deep sub bass, minimal chord stabs
[Hook] add melodic element — vocal chop, pluck riff, or arp
[Breakdown] strip to pads + percussion, slow filter sweep
[Build] reintroduce bass movement, gentle riser
[Hook] full groove returns with variation (new chop or filter position)
[Outro] elements exit one by one, hi-hat last
```

**Example Style Block:**
```
[Instrumental] Deep house, 122 BPM, warm analog pad, deep sub bass, shuffled hi-hats, hypnotic groove, late-night atmosphere, minimal stabs, filtered percussion, clean [No Vocals]
```

**Exclude Style:** `hard drop, aggressive, distorted bass, festival, harsh synth`

**Advanced Settings:** Weirdness 30-45%, Style Influence 60-75%

### Melodic Techno

**BPM:** 122-130 (sweet spot: 126)
**Key Instruments:** Driving kick, evolving arp sequence, lush reverb pad
**Mood:** Hypnotic, cinematic, emotional, dark yet beautiful, driving
**Avoid:** Cheesy melodies, pop chord progressions, acoustic drums, bright happy energy

**Structure Template:**
```
[Intro] kick + ride pattern, distant pad texture, sparse arp notes
[Build] arp sequence intensifies, add hi-hats, bass rumble grows
[Drop] full kick + bass lock, arp melody fully revealed, driving groove
[Breakdown] kick drops out, pads swell, arp goes sparse and reverb-heavy
[Build] kick returns gradually, arp rebuilds with new counter-melody
[Drop] second drop — denser arp pattern or added acid line
[Outro] elements dissolve, kick + ride continue, pad fades
```

**Example Style Block:**
```
[Instrumental] Melodic techno, 126 BPM, driving kick, evolving arp sequence, lush reverb pad, cinematic, hypnotic, dark atmosphere, analog warmth, clean mix, long reverb tails [No Vocals]
```

**Exclude Style:** `pop melody, acoustic guitar, bright, happy, cheesy`

**Advanced Settings:** Weirdness 40-55%, Style Influence 55-70%

### Ambient Electronic / Downtempo

**BPM:** 70-100 (sweet spot: 85)
**Key Instruments:** Granular pad, slow arp, textured field recordings
**Mood:** Atmospheric, contemplative, floating, dreamlike, immersive
**Avoid:** Four-on-the-floor kick, hard drops, aggressive bass, fast tempos, staccato rhythms

**Structure Template:**
```
[Intro] field recording or texture wash, single held note
[Verse] slow pad evolution, gentle arp emerges, soft sub bass
[Hook] melodic motif appears — piano note, pluck, or bell tone
[Breakdown] texture-only section, granular movement, no rhythm
[Build] rhythm returns softly — brushed percussion or glitchy clicks
[Hook] motif returns with harmonic shift or new texture layer
[Outro] all rhythm exits, pad sustains and fades to silence
```

**Example Style Block:**
```
[Instrumental] Ambient electronic, downtempo, 85 BPM, granular pad, slow arp, atmospheric, contemplative, dreamlike, soft sub bass, field recording texture, spacious mix, clean [No Vocals]
```

**Exclude Style:** `four-on-the-floor, hard drop, aggressive bass, fast, staccato`

**Advanced Settings:** Weirdness 45-65%, Style Influence 40-60%

### DnB / Breakbeat

**BPM:** 160-180 (sweet spot: 174)
**Key Instruments:** Amen break or fast breakbeat, reese bass, sharp snare
**Mood:** Intense, kinetic, urban, dark, rolling
**Avoid:** Slow tempo, four-on-the-floor kick, ambient pads, gentle mood, acoustic instruments

**Structure Template:**
```
[Intro] half-time beat or sparse percussion, bass hint, atmosphere
[Build] breakbeat speeds up or enters at full tempo, bass growl rises
[Drop] full breakbeat + reese bass, sharp snare hits, aggressive groove
[Breakdown] beat halves or drops, pad or vocal sample holds tension
[Build] snare roll or amen fill accelerates, bass warps upward
[Drop] second drop — different break pattern or bass rhythm variation
[Outro] beat simplifies to kick + snare, bass exits, ride continues
```

**Example Style Block:**
```
[Instrumental] Drum and bass, liquid DnB, 174 BPM, fast breakbeat, reese bass, sharp snare, rolling groove, dark atmosphere, intense energy, clean mix [No Vocals]
```

**Exclude Style:** `slow tempo, four-on-the-floor, ambient, gentle, acoustic`

**Advanced Settings:** Weirdness 35-55%, Style Influence 55-75%

### Chill Electronic (background)

**BPM:** 80-110 (sweet spot: 95)
**Key Instruments:** Soft pluck synth, warm pad, gentle sidechain pulse
**Mood:** Relaxed, warm, background-friendly, unobtrusive, steady
**Avoid:** Hard drops, aggressive bass, loud dynamics, complex arrangements, harsh frequencies

**Structure Template:**
```
[Intro] soft pad wash, gentle pulse, filtered texture
[Verse] pluck melody enters, light percussion — shaker or soft kick
[Hook] add warmth — layer a second pad or subtle bass movement
[Breakdown] return to pad-only, simplify to create breathing room
[Verse] melody returns with slight variation, same gentle energy
[Hook] fullest moment — all layers present but still soft
[Outro] pluck exits, pad sustains, percussion fades
```

**Example Style Block:**
```
[Instrumental] Chill electronic, lo-fi electronic, 95 BPM, soft pluck synth, warm pad, gentle sidechain, relaxed, background-friendly, clean mix, minimal arrangement [No Vocals]
```

**Exclude Style:** `hard drop, aggressive, loud, complex arrangement, harsh`

**Advanced Settings:** Weirdness 30-50%, Style Influence 50-70%

## Electronic Production Rules

Practical rules for getting clean electronic output from Suno.

### Subgenre Specificity

Suno responds strongly to precise subgenre labels. Small changes produce very different tracks:

| Vague Prompt | Specific Prompt | Result Difference |
|-------------|----------------|-------------------|
| techno | melodic techno | Adds arp melodies, emotional pads |
| techno | acid techno | Adds 303 squelch, harsher energy |
| techno | minimal techno | Strips to kick + hi-hat + one loop |
| house | deep house | Warmer, groovier, less aggressive |
| house | bass house | Heavier bass, wobble, more aggressive |
| electronic | ambient electronic | Removes beat emphasis, adds texture |

### Adjective Power

Small adjective changes reshape the entire track character:

| Adjective | Effect on Output |
|-----------|-----------------|
| dark | Minor key, heavier bass, moody atmosphere |
| hypnotic | Repetitive patterns, long loops, trance-like |
| minimal | Fewer elements, more space, subtlety |
| cinematic | Bigger reverb, orchestral hints, dramatic builds |
| clean | Better mix separation, less muddy output |
| raw | Grittier texture, less polished, more energy |
| warm | Analog-style saturation, softer highs, rounder bass |

**Always include `clean`** in Style blocks for electronic tracks — it noticeably improves mix quality and separation.

---

### Bass Design for Suno

Specify bass type explicitly. "Bass" alone is too vague for electronic.

| Bass Type | Description | Best For |
|-----------|-------------|----------|
| sub bass | Deep sine-wave low end, felt more than heard | Deep house, ambient, downtempo |
| acid bass | TB-303 squelchy resonant filter sweeps | Acid house, acid techno |
| reese bass | Detuned saw waves, thick and evolving | DnB, neurofunk, dark techno |
| 808 glide | Pitched 808 with portamento slides | Trap-influenced electronic, future bass |
| wobble bass | LFO-modulated dubstep bass | Dubstep, riddim, bass house |
| pluck bass | Short attack, quick decay, rhythmic | Tech house, minimal, micro house |
| FM bass | Metallic, digital, precise | IDM, glitch, experimental |

**Usage:** Write the specific bass type directly in the Style block — e.g., `reese bass` not just `bass`.

---

### Synth Design for Suno

Like bass, specify synth texture to guide Suno's sound choices:

| Synth Type | Description | Best For |
|-----------|-------------|----------|
| supersaw | Stacked detuned saw waves, huge and wide | EDM, trance, future bass drops |
| analog pad | Warm, evolving, filter-swept sustained tones | Deep house, ambient, downtempo |
| pluck | Short bright attack with fast decay | Tropical house, melodic techno, chill |
| saw wave | Single or few saw oscillators, raw and present | Techno, hard dance, leads |
| FM bell | Metallic, crystalline, bell-like overtones | IDM, ambient, melodic techno accents |
| square lead | Retro, chiptune-adjacent, buzzy | Synthwave, retro electronic |
| granular texture | Fragmented, evolving, unpredictable | Ambient, experimental, sound design |

**Usage:** Combine one synth type with one bass type for clear sonic direction — e.g., `analog pad, reese bass` or `pluck synth, sub bass`.

## Instrumental Electronic — Triple Layer Approach

For guaranteed vocal-free electronic output, use all three layers simultaneously:

### Layer 1: Style Block Tags
Place `[Instrumental]` at the **start** and `[No Vocals]` at the **end** of your Style block.

### Layer 2: Exclude Style
Set the Exclude Style (negative prompt) field to:
```
vocals, singing, humming, choir, voice
```

### Layer 3: Lyrics Field
Use **only structure tags** in the lyrics field — no words, no syllables:
```
[Intro]

[Build]

[Drop]

[Breakdown]

[Build]

[Drop]

[Outro]
```

### Why Triple Layer?

Single-layer approaches sometimes fail. Suno may still add vocals if you only use `[Instrumental]` in the Style block. The triple layer approach catches all three entry points where vocals can leak in:
- Style block tells Suno the intent
- Exclude Style actively suppresses vocal generation
- Empty lyrics field removes the text that would trigger singing

**If vocals still appear** (rare with triple layer), add `pure instrumental` to the Style block and `vocal melody, vocal harmonics` to the Exclude Style.

## Electronic Weirdness and Style Influence Guide

How to set Suno's Advanced Settings for different electronic styles. These interact — high Weirdness with low Style Influence lets Suno experiment freely, while low Weirdness with high Style Influence keeps output close to genre conventions.

| Electronic Style | Weirdness | Style Influence | Why |
|-----------------|-----------|-----------------|-----|
| Festival EDM / Big Room | 20-30% | 75-85% | Faithful genre recreation, predictable drops |
| Deep House / Tech House | 25-40% | 65-80% | Tight groove needed, some room for texture |
| Melodic Techno | 40-55% | 55-70% | Balanced — genre framework with room for Suno's interpretation |
| Trance / Progressive | 30-45% | 65-80% | Structured builds matter, moderate exploration |
| DnB / Breakbeat | 35-50% | 55-75% | Break patterns need consistency, bass can vary |
| Ambient / Downtempo | 45-65% | 35-55% | Texture variety benefits from higher Weirdness |
| Experimental / IDM | 65-85% | 25-45% | Let Suno explore, genre rules are loose |
| Chill Electronic | 30-50% | 50-70% | Gentle and steady, slight variation keeps it interesting |
| Synthwave / Retrowave | 25-40% | 70-85% | Retro accuracy matters, stay in the lane |
| Dubstep / Riddim | 35-55% | 55-75% | Bass design needs control, drops can vary |

### Reading the Table

- **Safe zone** (Weirdness < 35%, Style Influence > 70%): Predictable, genre-faithful output. Best for tracks that must sound "correct" in a genre.
- **Balanced zone** (Weirdness 35-55%, Style Influence 50-70%): Suno follows genre rules but adds its own ideas. Best default for most electronic styles.
- **Exploration zone** (Weirdness > 55%, Style Influence < 50%): Suno takes the wheel. Best for ambient, experimental, or when you want surprises.

**Vary across variations:** When generating 3 variations for a track, spread the settings across zones — e.g., one safe (30/75), one balanced (45/60), one exploratory (60/45). This gives the user a range from faithful to adventurous.
