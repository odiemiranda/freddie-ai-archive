# Special Techniques

## Call-and-Response: Vocal + Instrument

Vocal line immediately before instrument tag — instrument appears to "answer."

```
[Verse | raspy male vocal]
I've been running from the truth
[instrumental break saxophone]
You know I can't escape
[instrumental break saxophone]
```

Swap saxophone for guitar, violin, trumpet, harmonica as needed.

## Full Instrument Interaction (back-and-forth)

Different from simple call-and-response — full alternation between vocalist and instrument:

```
[Pre-Chorus | emotional build-up]
Tell me if you hear me now
[Guitar Solo | melodic response | blues scale]
Tell me if you feel it now
[Guitar Solo | melodic response | blues scale]
```

## Emotional Whiplash

Combine clashing emotions in a single line for dramatic effect:

```
[spoken word crying] Why did you leave me [laughter]
```

## ALL CAPS Emphasis Rules

- **1-3 words in ALL CAPS per section maximum**
- More than that dilutes the effect
- Works best on ONE key word per line

```
We rise together          ← weak
WE RISE together          ← correct
WE RISE TOGETHER NOW FOREVER  ← too much, loses impact
```

## Fictional Languages / Experimental Vocals

Suno can attempt phonetic generation of fictional or unusual languages:
- Klingon, Elvish, glossolalia, pure whistles, fantasy-style vocals, made-up chants
- Results are experimental — inconsistency across generations is normal
- Works better when genre supports it: `[epic | cinematic | fantasy]`
- Label as `[fictional language chant]` or `[elvish melody]` in lyrics

## Duet Format (Stable)

Style prompt: `This is a duet between [Name A] (male) and [Name B] (female), [genre], [other tags]`

```
[Duet: [Name A] male and [Name B] female]

[Verse 1]
[[Name A]]
[lyric line]

[Chorus]
[Both]
[lyric line]

[Verse 2]
[[Name B]]
[lyric line]
```
