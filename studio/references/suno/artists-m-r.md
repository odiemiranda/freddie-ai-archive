## M

**Mac DeMarco**: Slacker rock, jangly guitar, lo-fi warmth, laid-back male vocals
**Madonna**: Dance-pop, 80s synths, provocative energy, reinvention queen
**Marvin Gaye**: Soul, smooth male vocals, Motown production, sensual groove
**Massive Attack**: Trip-hop, dark atmospheric, Bristol sound, cinematic bass
**Megan Thee Stallion**: Southern female rap, Houston bounce, confident delivery
**Metallica**: Thrash metal, heavy riffs, aggressive energy, complex arrangements
**Metro Boomin**: Dark cinematic trap, heavy 808s, orchestral samples, producer tags
**MF DOOM**: Abstract hip-hop, dense wordplay, obscure samples, villain aesthetic
**Michael Jackson**: Pop perfection, funk bass, iconic hooks, 80s production
**Miles Davis**: Jazz trumpet, cool jazz, modal improvisation, Kind of Blue atmosphere
**Miley Cyrus**: Raspy female rock-pop, rebellious attitude, arena-ready
**Missy Elliott**: Innovative hip-hop, experimental production, futuristic beats

## Artist-Style Prompt Examples

### Radiohead Style
```
[Instrumental] Art rock, atmospheric and paranoid, glitchy electronic textures, melancholic piano, unconventional structure, OK Computer era [No Vocals]
```

### Prince Style
```
[Instrumental] Funk-rock, Minneapolis sound, funky synths, tight rhythm section, 80s production, sensual groove [No Vocals]
```

### Portishead Style
```
[Instrumental] Trip-hop, dark atmospheric, vintage samples, Bristol sound, cinematic strings, noir atmosphere [No Vocals]
```

### Nujabes Style
```
[Instrumental] Lo-fi hip-hop, jazz piano samples, boom-bap drums, warm bass, philosophical atmosphere, modal jazz influence, 85 BPM [No Vocals]
```
