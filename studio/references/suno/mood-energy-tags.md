## Energy Tags

| Tag | Intensity |
|-----|-----------|
| `[High Energy]` / `[Energy: High]` | Maximum intensity |
| `[Medium Energy]` / `[Energy: Medium]` | Moderate |
| `[Low Energy]` / `[Energy: Low]` | Minimal |
| `[Chill]` | Relaxed, laid-back |
| `[Driving]` | Propulsive, forward |
| `[Building]` / `[Energy: Building]` | Increasing tension |
| `[Explosive]` | Sudden intensity |
| `[Frantic]` | Chaotic, fast |
| `[Steady]` | Consistent energy |

**V5 extended format:** Natural language key:value tags are valid and give cleaner parsing:
- `[Mood: Euphoric]`
- `[Atmosphere: Cyberpunk]`
- `[Vibe: Midnight drive]`
