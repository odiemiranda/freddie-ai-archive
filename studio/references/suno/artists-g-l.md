## Artist-Style Prompt Examples

[Instrumental] Conscious hip-hop, jazz-influenced beats, complex rhythm, West Coast production, introspective atmosphere, storytelling mood [No Vocals]

```

```

```

### Lana Del Rey Style
```
[Instrumental] Cinematic dream pop, melancholic strings, vintage reverb, Hollywood noir atmosphere, orchestral arrangement [No Vocals]
