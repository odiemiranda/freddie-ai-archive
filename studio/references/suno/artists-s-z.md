## How to Use Artist References

1. **Find the artist** closest to your vision
2. **Extract key elements** from their description
3. **Combine with other elements** (mood, BPM, instruments)
4. **Add [Instrumental] and [No Vocals]** tags for background music

### Template:
```
[Instrumental] [Artist] style, [key element 1], [key element 2], [mood], [BPM if relevant] [No Vocals]
```
