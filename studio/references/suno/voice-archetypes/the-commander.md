# The Commander

> Deep authoritative baritone — epic leaders, war speeches, cinematic narration

---

## Voice Profile

| Attribute | Value |
|-----------|-------|
| **Register** | Deep baritone / bass |
| **Tone** | Warm, gravelly, resonant |
| **Delivery** | Measured, deliberate, commanding |
| **Energy** | Controlled power — never shouts, never whispers |
| **Pacing** | Slow and deliberate, dramatic pauses between phrases |

## Style Tags

```
deep resonant baritone male vocals, authoritative and commanding, warm gravelly tone, powerful and measured delivery
```

## Exclude Style

```
tenor, falsetto, female vocals, high-pitched, nasal, auto-tune, pop, electronic
```

## Lyrics Technique

### Vocal direction tags (use before specific lines)
```
[Spoken word, Deep baritone, Authoritative, Gravelly]    ← verses, monologue sections
[Powerful vocals, Deep baritone, Commanding]              ← chorus, inspirational lift
[Stripped back, Spoken word, Gravelly, Intimate]           ← bridge, quiet moments
[Spoken word, Deep baritone, Fading]                      ← outro, final words
```

### Writing rules
- **Spoken word heavy** — this voice speaks more than sings. Use `[Spoken word]` on most sections
- **Short declarative lines** — "Not to kneel. Not to fall." Fragments over sentences
- **Ellipses for dramatic pauses** — "Before the light... there was silence"
- **ALL CAPS for power moments** — "WE WILL NOT BREAK!" (sparingly, 1-2 per song)
- **Lines under 8 syllables** — keeps the delivery measured, prevents rushing
- **No rhyme chasing** — this voice doesn't need rhyme. Direct speech is stronger

### Line examples
```
Before the light... there was silence
And in that silence... we made a choice
Not to kneel. Not to fall.
But to rise... one more time
```

```
One voice... is enough
One step... is enough
The rest... will follow
```
