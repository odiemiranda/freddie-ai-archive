## BPM Psychology - Complete Guide

### BPM Ranges by Purpose

| Purpose | BPM Range | Why It Works |
|---------|-----------|--------------|
| **Deep Sleep** | 40-60 | Slows heart rate, mimics delta wave brain frequencies |
| **Meditation** | 60-75 | Matches relaxed breathing (12-15 breaths/min) |
| **Study/Focus** | 75-95 | Alert but not anxious, optimal cognitive state |
| **Coding/Work** | 90-110 | Maintains momentum without stress |
| **Morning Energy** | 110-130 | Motivates without cortisol spikes |
| **Workout/High Energy** | 130-150+ | Drives physical activity |

### BPM Variation Strategy

When generating 3+ variations, vary BPM strategically across the niche's range:

| Strategy | When to Use | Example (Lo-fi, range 75-90) |
|----------|-------------|------------------------------|
| **Low / Mid / High** | Default — covers the full range | 78, 83, 88 |
| **Tight cluster** | When the sweet spot matters more than range | 81, 83, 85 |
| **Anchor + explore** | One safe pick + two experiments | 82, 76, 90 |

**Rules of thumb:**
- Vary by at least ±3 BPM between variations (otherwise results sound identical)
- Stay within the niche's recommended range — jumping outside it changes the feel entirely
- Pair BPM changes with key changes for maximum variation (e.g., 78 BPM/C Minor, 83 BPM/A Minor, 88 BPM/D Minor)
- Suno approximates ±5 BPM, so don't stress about exact numbers — the range matters more than the digit

### Key Variation Strategy

When generating 3+ variations, vary keys alongside BPM for maximum tonal contrast.

| Strategy | When to Use | Example (Lo-fi) |
|----------|-------------|-----------------|
| **Relative minor/major** | Safe contrast — same notes, different feel | C Minor → E♭ Major → A♭ Major |
| **Parallel keys** | Same root, different mood (major ↔ minor) | C Minor → C Major → C Minor |
| **Circle-of-fifths neighbors** | Subtle shift, feels natural | C Minor → G Minor → D Minor |
| **Mood-matched keys** | Pick keys by target emotion (see Key Psychology below) | Nostalgic: A Minor / Sad: D Minor / Peaceful: F Major |

**Rules of thumb:**
- Don't use the same key for all 3 variations — it wastes the opportunity for tonal contrast
- Minor keys dominate lo-fi, study, and melancholic niches (C Minor, A Minor, D Minor)
- Major keys work better for morning, uplifting, spa, and meditation (C Major, F Major, G Major)
- Unusual keys (F♯ Minor, B♭ Minor) are less reliable in Suno — stick to common keys for consistent results
- When unsure, use the "mood-matched" strategy: pick each variation's key from the Key-by-Key Emotional Guide based on its specific mood angle

## Musical Key Psychology

### Major vs Minor

- **Major keys**: Happy, bright, uplifting (nucleus accumbens activation)
- **Minor keys**: Sad, mysterious, contemplative (amygdala engagement)

### Key-by-Key Emotional Guide

| Key | Emotional Quality | Best For |
|-----|-------------------|----------|
| **C Major** | Pure, neutral, calming | Sleep, meditation, universal |
| **C Minor** | Melancholic yet accessible | Lo-fi (most popular), study |
| **D Major** | Triumphant, bright | Morning, uplifting |
| **D Minor** | THE saddest key | Dark academia, deep emotion |
| **E Major** | Joyful, powerful | Energetic, anthems |
| **E Minor** | Mysterious, gothic | Dark academia, moody |
| **F Major** | Warm, pastoral | Healing, relaxation |
| **G Major** | Peaceful, natural | Nature sounds, gentle |
| **A Minor** | Contemplative, nostalgic | Lo-fi, reflection |
| **B♭ Major** | Sophisticated, jazzy | City pop, jazz |
| **E♭ Major** | Bright, uplifting | City pop, funk |

### Industry Secrets

- 80% of viral lo-fi uses **C Minor** or **A Minor**
- Dark academia gravitates to **D Minor** and **E Minor**
- City Pop loves **E♭ Major**, **F Major**, **B♭ Major**
- Sleep music stays in **C Major** or **F Major** (non-threatening)

### Complete Lo-Fi Structure Example

```
[Intro] [Instrumental] Soft vinyl crackle, gentle Rhodes piano, C Minor, 80 BPM, atmospheric
[Build] Add soft boom-bap drums, warm sub bass enters gradually
[Hook] [Instrumental] Full lo-fi arrangement, dusty drums, melancholic piano melody, vinyl texture, nostalgic atmosphere
[Break] Drums drop out, just piano and gentle rain sounds
[Hook] Return to full beat, reinforcing the main theme
[Outro] [Fade Out] Piano fades gently, rain continues softly [No Vocals]
```

### Complete Sleep Music Structure

```
[Intro] [Instrumental] Deep sub bass drone, 40 BPM, C Major, delta wave frequencies
[Ambient] Soft synth pad layers, gentle ocean waves texture, extremely slow, peaceful, dreamlike [No Vocals]
[Sustained] Minimal variation, continuous soothing tones, therapeutic frequencies

```

## Song Duration & Extend Strategies

### How Duration Works in Suno

Suno generates tracks of varying length depending on lyrics and structure. Understanding this helps you control output.

| Factor | Effect on Duration |
|--------|-------------------|
| **More lyrics sections** | Longer song — each section adds time |
| **Repeated choruses** | Adds 20-40 sec per repeat |
| **`[Instrumental Break]` / `[Solo]`** | Adds 15-30 sec per tag |
| **`[Sustained]`** | Extends current section significantly |
| **`[Fade Out]`** | Adds 15-30 sec fade tail |
| **Fewer sections** | Shorter song — minimal lyrics = ~1-2 min |
| **`[End]`** | Forces an abrupt stop |

### Duration Targets

| Target | Strategy |
|--------|----------|
| **Short (~1-2 min)** | Intro → Verse → Hook → Outro. Minimal sections, no repeats |
| **Standard (~2-3 min)** | Intro → Verse → Chorus → Verse → Chorus → Outro. The default pop structure |
| **Extended (~3-4 min)** | Add Bridge, second Hook, Instrumental Break, repeat Chorus with variation |
| **Long (~4+ min)** | Multiple verses, Bridge, Solo, repeated Choruses, `[Sustained]` sections, or use Extend |

### Writing Lyrics for Length

**For shorter tracks:** Keep lyrics sparse, 2-3 sections total
```
[Intro]
[Soft piano enters]

[Hook]
[Full arrangement, nostalgic]

[Outro]
[Fade with rain]
```

**For longer tracks:** Add more sections and repeats
```
[Intro]
[Vinyl crackle, piano]

[Verse 1]
[Soft drums join, mellow bass]

[Hook]
[Full arrangement]

[Break]
[Just piano and rain]

[Verse 2]
[Subtle variation, new melody]

[Hook]
[Return to full beat]

[Instrumental Break]
[Solo piano moment]

[Hook]
[Final repeat, slight build]

[Outro]
[Fade Out]
```

### Using Suno's Extend Feature

After generating a track, Suno's **Extend** button continues from where the song ended. Tips:

- **Plan for extend** — end your lyrics at a natural continuation point, not `[End]` or `[Fade Out]`
- **Leave the last section open** — a `[Hook]` or `[Sustained]` ending is easier to extend than a resolved `[Outro]`
- **Match the energy** — when extending, Suno continues from the ending mood. A quiet ending extends quietly
- **For ambient/lo-fi loops** — use `[Sustained]` as your final tag. This tells Suno to maintain energy, making extend seamless
- **For songs with vocals** — end mid-story (after a Chorus, before the final Outro) so the extend can add a Bridge or final Chorus

### Structure Templates by Duration

**2-minute lo-fi loop:**
```
[Intro] → [Hook] → [Break] → [Hook] → [Sustained]
```

**3-minute song:**
```
[Intro] → [Verse 1] → [Chorus] → [Verse 2] → [Chorus] → [Bridge] → [Chorus] → [Outro]
```

**4+ minute epic:**
```
[Intro] → [Verse 1] → [Pre-Chorus] → [Chorus] → [Verse 2] → [Chorus] → [Instrumental Break] → [Bridge] → [Final Chorus] → [Outro] → [Fade Out]
```

### Named Structure Patterns

Additional structural archetypes from production vocabulary that complement the duration-based templates above.

| Pattern | Structure | Best For |
|---------|-----------|----------|
| **Song (AABA)** | `[Verse] → [Verse] → [Bridge] → [Verse]` | Jazz standards, classic songwriting, ballads |
| **Song (ABABCB)** | `[Verse] → [Chorus] → [Verse] → [Chorus] → [Bridge] → [Chorus]` | Extended pop/rock songs with contrast |
| **Instrumental (Theme + Variation)** | `[Theme] → [Variation 1] → [Variation 2] → [Theme Reprise]` | Cinematic, classical-influenced, ambient |
| **Trailer (Rise/Hit/Rise/Hit)** | `[Build] → [Drop/Hit] → [Build] → [Final Hit]` | Trailer music, cinematic tension, epic moments |

## Production Descriptors

### How to Use Descriptors

Pick **1-2 words per category** — more than that dilutes the signal. A good Style prompt uses 3-5 total descriptors drawn from different categories to create a coherent sonic picture.

| Category | Use For | Pick |
|----------|---------|------|
| Texture | How the sound "feels" physically | 1-2 words |
| Space | How big/intimate the room is | 1 word |
| Era | When the sound belongs | 1 word or phrase |
| Energy | How the track moves | 1 word |
| Atmosphere | The scene/setting the music evokes | 1-2 words |

**Avoid conflicts:** Descriptors from the same category can contradict each other. Don't combine:
- `warm` + `cold`, `crisp` + `fuzzy`, `raw` + `polished` (texture)
- `spacious` + `intimate`, `dry` + `reverb-heavy` (space)
- `driving` + `laid-back`, `explosive` + `restrained` (energy)

**Good combinations** pull from different categories:
- `dusty` (texture) + `intimate` (space) + `vintage` (era) + `laid-back` (energy) + `cozy` (atmosphere)
- `crisp` (texture) + `spacious` (space) + `modern` (era) + `driving` (energy) + `cinematic` (atmosphere)

### Texture Words
```
lo-fi, crisp, polished, raw, warm, cold, fuzzy, clean, gritty, lush, sparse, dense, dusty, grainy, saturated, analog, digital, vintage, modern
```

### Space Words
```
dry, reverb-heavy, spacious, intimate, cavernous, tight, wet, ambient, echo-laden, distant, close, expansive, claustrophobic
```

### Era References
```
vintage, retro, modern, futuristic, 60s, 70s, 80s, 90s, 2000s, analog, digital, tape-era, vinyl-era, MTV-era, golden age
```

### Energy Words

## The Correct vs Incorrect Format

### INCORRECT (Vocals Likely):
```
Lo-fi hip hop with piano and drums
```

### CORRECT (100% Instrumental):
```
[Instrumental] Lo-fi hip hop, dusty Rhodes piano, soft vinyl drums, warm bass, melancholic, 85 BPM, C Minor [No Vocals]
```

### Why It Works:
- `[Instrumental]` opens the prompt
- All descriptive elements in the middle
- `[No Vocals]` closes the prompt
- This "bracketing" dramatically reduces vocal hallucinations
