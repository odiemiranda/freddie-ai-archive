# Instrument Bracket Tags

Quick-lookup for `[Instrument:]` tag values that Suno recognizes. Use these inside typed bracket `[Instrument:]` tags or standalone bracket tags.

For detailed instrument descriptions, adjectives, and production guidance, see `instruments.md`.
