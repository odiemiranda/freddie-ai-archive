# Ambient & Soundscape Niche Prompts

Battle-tested prompts for ASMR, nature ambience, rain, and weather soundscapes.

> **Note:** Some example prompts below use an older format with `[Intro]`/`[Build]` tags inline. The current best practice is: **Style block = sonic palette only** (genre, instruments, mood, BPM, key, texture). Structure tags go in the **Lyrics block**. Adapt these examples accordingly.

---

## ASMR & SOUNDSCAPES

### Rain & Thunder

**Core Characteristics:**
- Advanced Settings: Weirdness 40-60%, Style Influence 40-60% — balanced; ASMR benefits from organic unpredictability. Vary: 40/60, 50/50, 60/40

**Prompts:**

```
[Instrumental] Gentle rain ambience, soft rainfall on window, distant thunder rolls, no music, pure rain sounds, deeply relaxing, sleep inducing, natural ASMR [No Vocals] [Ambient]
```

```
[Instrumental] Thunderstorm sounds, heavy rain, rolling thunder, lightning ambience, no music, storm soundscape, powerful yet calming, natural sleep aid [Sustained] [No Vocals]
```

```
[Instrumental] Rain on tent, camping ambience, gentle rainfall, soft wind, no music, cozy shelter sounds, deeply soothing, outdoor ASMR [No Vocals]
```

**Core Characteristics:**
- BPM: No tempo or 50-65 (pure soundscape — rhythm would break immersion)
- Keys: Not applicable for pure ASMR; A Minor or C Major if adding soft music underneath
- Elements: Natural sounds — rain, birds, water, wind, fire, ambient noise
- Atmosphere: Immersive, non-musical, deeply calming, ASMR

**Natural-Language Prompts:**

```
ASMR rain soundscape, heavy rainfall, distant thunder, 55 BPM, A Minor, cozy indoor atmosphere
```

```
Forest ASMR, birds chirping, leaves rustling, gentle stream, 60 BPM, G Major, early morning nature sounds
```

```
Ocean ASMR, gentle waves on shore, soft sea breeze, distant seagulls, 55 BPM, C Major, peaceful beach mood
```

```
Garden ASMR, bees buzzing softly, light breeze, bird songs, 60 BPM, G Major, sunny afternoon outdoor mood
```

```
Japanese garden ASMR, water trickling, bamboo sounds, wind chimes, 55 BPM, A Minor, zen and meditative mood
```

### Fireplace & Cozy

**Core Characteristics:**
- BPM: None (pure soundscape — no rhythm)
- Keys: Not applicable (non-musical)
- Elements: Crackling fire, wood pops, warm ambience
- Atmosphere: Cozy, safe, winter comfort, deeply warm

**Prompts:**

```
[Instrumental] Crackling fireplace, warm fire sounds, soft wood pops, cozy cabin ambience, no music, pure fire ASMR, deeply comforting, winter relaxation [No Vocals] [Ambient]
```

```
[Instrumental] Fireplace and rain, crackling fire, gentle rainfall outside, cozy winter night, no music, combined ASMR ambience, deeply soothing comfort [Sustained] [No Vocals]
```

### Cafe & Urban

**Core Characteristics:**
- BPM: None (pure soundscape — no rhythm)
- Keys: Not applicable (non-musical ASMR)
- Elements: Soft chatter, espresso machine, cup sounds, typing, page turning
- Atmosphere: Productive, urban, focused, cozy background

**Prompts:**

```
[Instrumental] Coffee shop ambience, soft chatter, espresso machine, cup clinking, background murmur, no music, cafe ASMR, productive atmosphere [No Vocals] [Ambient]
```

```
[Instrumental] Library ambience, soft page turning, distant typing, hushed whispers, no music, study hall ASMR, focused concentration atmosphere [No Vocals]
```

## NATURE AMBIENT

### Forest & Woodland

**Core Characteristics:**
- BPM: 55-75 (slow, grounding)
- Keys: G Major, D Major, C Major
- Elements: Bird songs, rustling leaves, flowing water, soft wind
- Atmosphere: Peaceful, grounding, immersive, organic
- Advanced Settings: Weirdness 40-60%, Style Influence 50-70% — organic sounds benefit from moderate variation. Vary: 40/65, 50/55, 60/45

**Tagged Prompts:**

```
[Instrumental] Mountain stream ambient, flowing water, distant bird songs, light wind, peaceful wilderness, G Major, 60 BPM, natural grounding, organic calm [No Vocals] [Ambient]
```

```
[Instrumental] Autumn forest walk, crunching leaves, distant woodpecker, cool breeze, nostalgic and peaceful, D Major, 65 BPM, seasonal tranquility [No Vocals] [Ambient]
```

```
[Instrumental] Spring meadow ambient, light breeze, birds singing, cheerful and fresh outdoor mood, C Major, 70 BPM, pastoral warmth [No Vocals] [Ambient]
```

```
[Instrumental] Japanese bamboo forest, wind through bamboo, soft rustling, peaceful and meditative, G Major, 55 BPM, zen garden serenity [No Vocals] [Ambient]
```

**Natural-Language Prompts:**

```
Mountain stream ambient, flowing water, distant birds, light wind, 60 BPM, G Major, peaceful wilderness mood

```
Tropical rainforest, exotic birds, rain on large leaves, insects, 65 BPM, D Major, lush and vibrant atmosphere

```
Autumn forest walk, crunching leaves, distant woodpecker, cool breeze, 65 BPM, D Major, nostalgic and peaceful

```

## RAIN & WEATHER AMBIENCE

### Rain Variations

**Core Characteristics:**
- BPM: 55-75 (or no tempo — pure ambience)
- Keys: A Minor, D Minor, C Minor (melancholic rain keys)
- Elements: Rain patterns, thunder, indoor ambience, optional soft music
- Atmosphere: Cozy, safe, meditative, hypnotic
- Advanced Settings: Weirdness 30-50%, Style Influence 50-70% — rain ambience should feel natural, not jarring. Vary: 30/65, 40/55, 50/50

**Tagged Prompts:**

```
[Instrumental] Heavy rain on window, cozy apartment ambience, soft lo-fi music underneath, warm and safe, A Minor, 70 BPM, rainy day comfort [No Vocals] [Ambient]
```

```
[Instrumental] Thunderstorm with music, dramatic rain, rolling thunder, soft piano underneath, powerful and cozy, D Minor, 60 BPM, storm shelter warmth [No Vocals] [Sustained]
```

```
[Instrumental] Rain on rooftop, steady rhythm, meditative quality, hypnotic and grounding, C Minor, 65 BPM, continuous rainfall comfort [No Vocals] [Ambient]
```

```
[Instrumental] Winter rain, cold grey day, slow drops, melancholic piano, introspective atmosphere, A Minor, 55 BPM, window reflection [No Vocals] [Fade Out]
```

```
[Instrumental] Midnight rain, quiet city, soft rainfall, peaceful solitude, reflective and calm, D Minor, 60 BPM, late night rain [No Vocals] [Ambient]
```

**Natural-Language Prompts:**

```
Heavy rain on window, soft lo-fi music underneath, 70 BPM, A Minor, cozy apartment, warm and safe
```

```
Gentle spring rain, soft patter on leaves, birds between drops, 65 BPM, G Major, peaceful
```

```
Thunderstorm with music, dramatic rain, rolling thunder, soft piano, 60 BPM, D Minor, powerful and cozy
```

```
Rain in the city, puddles, traffic in distance, 65 BPM, A Minor, night atmosphere, melancholic and cinematic
```

```
Rain on tent camping, steady rainfall, 60 BPM, C Minor, outdoors safe feeling, cozy and adventurous
```

```
Tropical rainstorm, intense downpour, humid atmosphere, 70 BPM, D Minor, dramatic and wild nature sound
```

```
Rain on rooftop, steady rhythm, meditative quality, 65 BPM, C Minor, hypnotic and grounding
```

```
Winter rain, cold grey day, slow drops, melancholic piano, 55 BPM, A Minor, introspective atmosphere
```

```
Rain cafe ambience, rain outside window, warm inside, 68 BPM, G Major, quiet conversations, cozy mood
```

```
Midnight rain, quiet city, soft rainfall, 60 BPM, D Minor, peaceful solitude, reflective and calm
```

> **Tip:** When generating 3+ variations, vary BPM across the 55-75 range and pair with different keys (A Minor, D Minor, C Minor for melancholic rain; G Major, C Major for cozy rain). See `production.md`.
