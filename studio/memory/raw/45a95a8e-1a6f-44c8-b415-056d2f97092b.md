# Session 45a95a8e-1a6f-44c8-b415-056d2f97092b

**Shirozen** — the cognitive brain engine for Freddie AI. We created the project brief last session.

The core idea: the current system is **reactive** (agents query brain.db when asked, reflect after workflows, consolidate when inbox fills). Shirozen makes it **proactive** — it observes, anticipates, and pushes the right context before you ask.

**Six modules:**
1. **Provenance Tracker** — every knowledge entry traces to its source raw session
2. **Contradiction Detector** — flags conflicting knowledge instead of silently merging
3. **Decision Feedback Loop** — outcomes flow back to update confidence scores
4. **Pattern Detector** — extracts recurring workflow fingerprints from raw transcripts
5. **Context Streamer** — multi-transport (HTTP/stdio/WebSocket) push of predicted context
6. **Working Memory** — warmer layer between conversation context and brain.db cold storage
