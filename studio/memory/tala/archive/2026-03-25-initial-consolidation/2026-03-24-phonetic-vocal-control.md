Suno's dictionary is trained on clean standardized English. When it recognizes a word, it pulls a preset pronunciation — polished, generic, American. To get authentic regional/character voices, deliberately misspell words phonetically so Suno is forced to render sounds manually instead of using presets.

**Triple reinforcement (all three must agree):**
1. Style box — genre + accent + vocal character
2. Metadata header — voice directive at top of Lyrics box
3. Phonetic lyrics — misspelled words forcing the pronunciation

Without all three, Suno compromises back to its generic default.

**Spelling = musical notation:**
- Stretched vowels (`na-ame`) = longer notes
- Hard consonants (`Da wata`) = rhythmic punch on beat
- Open vowels (`awaa`) = warmer, more open tone
- Dropped endings (`walkin'`) = casual/regional feel

**Accent recipes in vocals.md:** Scottish, Jamaican, Southern Gothic, Irish, Cockney — each with Style box, header, and phonetic patterns.

**Workflow:** Write lyrics standard → use ChatGPT to translate phonetically for target accent → paste with metadata header.

**Stacks with range headers:** Range controls pitch (what notes). Phonetics control pronunciation (what mouth shape). Together = full vocal director stack.
