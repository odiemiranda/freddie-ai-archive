## Ensemble / Layer Framework

Two building blocks for every Lyrics section. Pipe (`|`) always means "play together."

### Ensemble
- The instrumental bed. First line of every section. Pipe syntax.
- `[Verse | shamisen jazz phrases | soft harp underneath | brushed drums lazy groove]`
- Default for ALL tracks — instrumental and vocal.
- Suno reads one instruction = instruments collaborate.

### Layer
- Intentional additions on top of the ensemble. Separate bracket lines within the block.
- Layers can also use pipes: `[Guitar phrase | single pluck]`
- Types: instrumental breaks, piped instrument layers, vocal techniques, ad-libs, effects

### Rules
- Instrumental tracks = ensemble lines only. No layers.
- Vocal tracks = ensemble + lyrics + selective layers.
- Ensemble line always comes first in every section.
- Each layer is a deliberate choice — don't over-stack.
- Separate instrument lines WITHOUT lyrics = competing layers (bad). Always use pipe syntax for instruments that should play together.
