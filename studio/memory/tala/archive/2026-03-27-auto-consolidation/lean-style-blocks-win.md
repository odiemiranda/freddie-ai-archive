## Discovery: Style Block Length vs Audio Quality

**Rule: Target 140-170 chars for instrumental BGM. Previous 250 target was too high.**

### Evidence
- 291 chars → chaotic output, instruments competing, piano hallucination at 75% mark
- 248 chars → better but still some chaos
- 161 chars → clean separation, instruments jive together
- 145 chars → smoothest output, zen-like

### What to cut first
1. Chord progressions — they invite instruments to "interpret" the chords (especially piano)
2. Textures (tape warmth, rolled-off highs) — genre implies these
3. "no melody", "no fills" — negative constraints confuse more than help. Use positive: "pad chords", "sparse"
4. [Background Music] tag — unnecessary if arrangement already implies it

### What to keep (never cut)
- Genre (single word: "Jazz" not "Jazz fusion")
- Lead instrument + one behavior word
- Support instrument + one role word
- Rhythm instrument + one character word
- Key, BPM
- "clean mix, separated instruments"
- [Instrumental] and [No Vocals]
