Fewer ingredients = clearer signal = better Suno output. The track "two-tabs-left-open" (3 instruments, 1 genre, simple lyrics) worked perfectly. Tracks with 4-5 instruments, 3-4 genres, and 6-8 mood words sounded out of sync with no harmony.

**Why:** Suno gets confused when given too many simultaneous directions. Multiple genre labels pull in different directions. Multiple plucked string instruments in overlapping registers fight each other. Style blocks that read like spec sheets instead of scenes produce worse results.

**How to apply:** Enforce hard limits: max 2 genres, max 3 instruments, max 1 rhythm source, max 5 mood descriptors, max 3 lyrics direction lines per section. Style blocks should read like a scene, not a shopping list.
