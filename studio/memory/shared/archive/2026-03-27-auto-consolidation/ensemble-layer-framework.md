## Shared: Ensemble/Layer Framework

For all Suno tracks: pipe = ensemble (play together), separate bracket line = layer (stack on top).

- **Ensemble**: `[Verse | lead + action | support + position | rhythm + feel]` — always first line
- **Layer**: `[instrumental break saxophone]` or `[Guitar phrase | single pluck]` — inserted between lyrics
- Instrumental tracks: ensemble only. Vocal tracks: ensemble + lyrics + layers.
- Piped elements within a layer also play together.
