Never fabricate words in non-English languages to force a rhyme. If you're writing in Tagalog, Filipino, or any language outside English, verify the word is real before using it. Use slant rhyme or rewrite the line instead of inventing vocabulary.

**Why:** Rune wrote "Kulang sa sayin" — "sayin" is not a Tagalog word. It was fabricated to rhyme with "pa rin." The user caught it. This is worse than a bad rhyme — it breaks trust in the lyrics.

**How to apply:** When writing in any non-English language, prefer common conversational words you're confident about. If unsure whether a word exists, restructure the line. Rhyme tax that costs authenticity is never worth it. Also avoid brand names (e.g., Jollibee) — use generic references instead.
