## Decisions

1. **Music cards as unified format** — one card type with a `type` field, not separate mix/vocal card types. Vocal sections are optional, present when type has vocals.
2. **Freddie memories in vault** — moved from auto-memory dir to `vault/studio/memory/freddie/` so all memories live together, synced via iCloud, backed up to archive. MEMORY.md stays as index.
3. **Vocal range headers mandatory** — Tala must use note-range headers on all vocal tracks. Adjectives are opinions, note values are facts.
4. **Audio gitignored from archive** — music card audio clips too large for git. iCloud is the sole sync for audio files.

## Learned

1. **"Deep" is an opinion, E2 is a fact** — Suno's vocal register is controlled by note-range headers in the Lyrics box, not adjectives in the Style box. Without exclusion lists, Suno drifts to tenor-pop on choruses.
2. **Phonetic misspelling bypasses Suno's dictionary** — when Suno hits a word it doesn't recognize, it stops reading and starts rendering. This forces authentic accents and controls note duration via vowel spelling.
3. **Suno models physical vocal mechanics** — vocal cord flips (yodeling), two-tone harmonics (throat singing), and whistle register all work because Suno simulates the physics, not just the sound. But it has a hard melodic bias — can't do non-musical vocalization.
4. **Audio Influence is the missing third slider** — controls similarity to uploaded reference audio. Connects music cards directly to generation (upload clip → set AI% → groove-guided output).
