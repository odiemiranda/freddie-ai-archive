## Learned

- Pipe = ensemble (instruments together), separate bracket line = layer (stacks/competes)
- Lean Style blocks (140-170 chars) beat verbose (250+) for instrumental
- ~14 focused instrument excludes beat 30 noisy ones
- Chord progressions in Style invite piano hallucination — drop by default
- "Jazz" beats "Jazz fusion" — simpler genre = cleaner output
- Scene/atmosphere cues improve coherence ("morning air", "stillness")
- Support instrument gets solo sections, not shared dominance
- LLMs cannot count characters accurately — charcount tool provides ground truth
- Consolidation tool's maxOutputTokens was too low (8192) — silently lost 3 of 4 knowledge files
