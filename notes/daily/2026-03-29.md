## Notes

- **Test result — Bracket format comparison (Suno v5.5 base):** Tested three bracket formats in a single generation — typed brackets (multi-line), single-line stacking, and pipe-stacked sections. All produce equivalent output. No audible difference in quality or interpretation. Conclusion: bracket format is a human readability choice, not a Suno quality factor. Style: "Jazz Lo-fi Celtic Shamisen Fusion, Warm Tape, 72 BPM, D Minor", W30/SI70.
- **Memory Resilience — all 4 features implemented:**
1. **Confidence decay** — `computeDecayedConfidence()` in brain/queries.ts. Full strength 30 days, linear decay to 0.3 floor at 90 days. All decision queries now show raw + decayed confidence.
2. **Contradiction tracking** — reflect.ts detects overlapping knowledge via brain.db FTS5 before writing inbox files. Contradictions flagged in frontmatter `contradicts:` field. Per-agent and shared scope.
3. **Strategic compaction reminder** — PostToolUse counter hook, Stop hook checks thresholds (50+ gentle, 80+ strong). Counter resets on SessionStart.
4. **PreCompact state save** — Saves tool call count, compaction count, timestamps to out/session-state.json. Tracks multiple compactions per session.
- **Decision:** Deferred Bifrost gateway setup. Wick starts on Sonnet. Bifrost + MiniMax M2.7 is the upgrade path after dev agent ecosystem is stable. Build order: Phase 0 → 1 → 2 → 3 → Bifrost.
- **Finding: ChatGPT Suno GPT analysis.** It's a Custom GPT (system prompt only, no code backend). Quality comes from accumulated prompt craft: acting directions in brackets, SFX-as-instruments in Style, prose-style vibe paragraphs for character genres. Our typed bracket system is stronger for production tracks, but comedy/novelty genres benefit from looser, more performative formatting. Key insight: code-driven prompt construction (genre templates, acting direction libraries) would give agents a better starting point than AI-from-scratch each time.
